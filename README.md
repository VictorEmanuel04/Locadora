# Resumo

O presente trabalho tem como objetivo o desenvolvimento de uma aplicação web de locadora de filmes inspirada em plataformas modernas de streaming. O sistema buscará unir funcionalidades tradicionais de aluguel de filmes com recursos de interação entre usuários, proporcionando uma experiência mais dinâmica e moderna.

A aplicação contará com funcionalidades como catálogo de filmes, sistema de locação digital, histórico de filmes alugados, avaliações, resenhas, lista de desejos e recomendações simples baseadas nas preferências do usuário. Além disso, haverá uma área administrativa destinada ao gerenciamento do catálogo, preços e promoções.

O projeto será desenvolvido utilizando arquitetura web full-stack, com frontend em React, backend em Node.js e banco de dados relacional.

---

# 1. Tema

O trabalho final tem como tema o desenvolvimento de uma plataforma web de locadora de filmes moderna, inspirada em serviços de streaming digitais. O sistema permitirá que usuários realizem o aluguel de filmes de forma online, acompanhem seu histórico de locações e interajam com outros usuários por meio de avaliações, resenhas e recomendações.

---

# 2. Escopo

Este projeto terá as seguintes funcionalidades:

- Cadastro e autenticação de usuários;
- Listagem de filmes com filtros, busca e paginação;
- Visualização detalhada dos filmes;
- Sistema de aluguel digital de filmes;
- Controle do período de disponibilidade do aluguel;
- Carrinho de compras e simulação de pagamento;
- Wishlist/lista de desejos;
- Histórico de locações;
- Sistema de avaliações e resenhas;
- Sistema simples de recomendação baseado em gêneros mais assistidos ou mais bem avaliados;
- Painel do usuário com separação de filmes disponíveis e expirados;
- Opção de renovação/recompra de filmes expirados;
- Área administrativa para:
  - Cadastro de filmes;
  - Edição e remoção de filmes;
  - Controle de preços;
  - Gerenciamento de promoções.

---

# 3. Restrições

Neste trabalho não serão considerados:

- Processamento de pagamentos reais;
- Streaming real de vídeos;
- Sistema avançado de recomendação utilizando inteligência artificial;
- Chat em tempo real entre usuários;
- Integração com plataformas externas de pagamento;
- Aplicações mobile nativas.

Algumas funcionalidades poderão ser simuladas para fins acadêmicos, como o fluxo de pagamento e a disponibilidade dos filmes.

---

# 4. Protótipo

Protótipos iniciais das páginas do sistema foram elaborados para representar a interface visual da aplicação. Entre as páginas previstas estão:

- Página inicial com destaque de filmes;
- Catálogo de filmes;
- Página de detalhes do filme;
- Tela de login e cadastro;
- Carrinho de aluguel;
- Perfil do usuário;
- Painel administrativo.

O frontend será desenvolvido utilizando React e Material UI, buscando uma interface moderna inspirada em plataformas de streaming.
